Usage

To use these cards in your GitHub README or website, simply use the URL as the source of an image:

<img src="https://cp-card.vercel.app/card/cf/your-handle/profile" alt="Codeforces Profile" />
Note: Replace your-handle with your actual Codeforces or CodeChef handle.

Features

  • Real-time data from Codeforces and CodeChef APIs
  • Responsive SVG cards that look great at any size
  • Automatic color theming based on user rank
  • Detailed statistics including ratings, ranks, and problem counts
  • Interactive rating graphs with tooltips
  • Activity heatmaps showing submission patterns

Rate Limits

To ensure service stability, please respect the following rate limits:

  • Maximum 100 requests per minute per IP
  • Cards are cached for 5 minutes to reduce API load
